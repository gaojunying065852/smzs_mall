### 微信登录接口测试

**接口**: `POST /api/auth/wx-login`

**测试命令**:
```bash
curl -X POST http://localhost:8080/api/auth/wx-login \
  -H "Content-Type: application/json" \
  -d '{
    "code": "test_code_123456",
    "nickName": "测试用户",
    "avatarUrl": "https://example.com/avatar.jpg"
  }'
```

**预期响应**:
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "Bearer 1741098116789",
    "user": {
      "id": 2,
      "phone": null,
      "openId": "openid_test_code_123456",
      "nickname": "测试用户",
      "avatar": "https://example.com/avatar.jpg",
      "gender": 0,
      "birthday": null,
      "status": 1,
      "createdAt": "2026-03-04T17:41:56",
      "updatedAt": "2026-03-04T17:41:56"
    }
  }
}
```

**注意事项**:
1. code字段为必填项
2. nickName和avatarUrl为选填项
3. 如果用户已存在，会更新用户的昵称和头像信息
4. 返回的token为模拟生成，实际应使用JWT
